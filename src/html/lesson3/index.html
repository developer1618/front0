Элементы форматирования HTML

Элементы форматирования были предназначены для отображения особых типов текста:


  • <b> - Жирный текст
  • <strong> - Важный текст
  • <i> - Курсивный текст
  • <em> - Выделенный текст
  • <mark> - Выделенный текст
  • <small> - Меньший текст
  • <del> - Удалён текст
  • <ins> - Вставленный текст
  • <sub> - Подстрочный текст
  • <sup> - Надстрочный текст

На протяжении 60 лет WWF помогает людям и природе процветать. WWF, ведущая природоохранная организация в мире, работает почти в 100 странах. HTML <q> для коротких цитат

<p>Целью WWF является:

Целью WWF является: <q>Постройте будущее, в котором люди будут жить в гармонии с природой.</q></p>



HTML <abbr> для сокращений

<p>The <abbr title="Всемирная организация здравоохранения">ВОЗ</abbr> была основана в 1948 году.</p>

HTML <address> для контактной информации

<address>
Автор: Иван Иванов<br>
Посетите нас по адресу:<br>
ADMIN.TJ<br>
Душанбе<br>
Таджикистан
</address>

HTML <cite> для названия работы

<p><cite>Крик</cite> Эдвард Мунк. Написан в 1893 году.</p>

HTML <bdo> для двунаправленного переопределения

<bdo dir="rtl">Этот текст будет написан справа налево.</bdo>

Домашние задания: Форматирование и элементы HTML

  1. Создайте HTML-документ, включающий жирный, курсивный текст и подстрочные/надстрочные символы.
  2. Отобразите важные тексты, выделенные тексты и текст с меньшим шрифтом.
  3. Используйте цитирование с элементами <blockquote> и <q>, а также аббревиатуру с элементом <abbr>.
  4. Предоставьте контактную информацию с использованием элемента <address>.
  5. Отобразите название работы и текст, написанный справа налево с использованием элементов <cite> и <bdo>.
