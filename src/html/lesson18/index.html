Типы ввода HTML

Вот различные типы ввода, которые вы можете использовать в HTML:

  • <input type="button">
  • <input type="checkbox">
  • <input type="color">
  • <input type="date">
  • <input type="datetime-local">
  • <input type="email">
  • <input type="file">
  • <input type="hidden">
  • <input type="image">
  • <input type="month">
  • <input type="number">
  • <input type="password">
  • <input type="radio">
  • <input type="range">
  • <input type="reset">
  • <input type="search">
  • <input type="submit">
  • <input type="tel">
  • <input type="text">
  • <input type="time">
  • <input type="url">
  • <input type="week">

Тип ввода Текст

Элемент HTML <form> используется для создания HTML-формы для ввода данных пользователем:

<input type="text"> определяет однострочное поле ввода текста :

<form>
  <label for="fname">First name:</label><br>
  <input type="text" id="fname" name="fname"><br>
  <label for="lname">Last name:</label><br>
  <input type="text" id="lname" name="lname">
</form>


Тип ввода Пароль

<input type="password"> определяет поле пароля :

<form>
  <label for="username">Username:</label><br>
  <input type="text" id="username" name="username"><br>
  <label for="pwd">Password:</label><br>
  <input type="password" id="pwd" name="pwd">
</form>


Тип ввода Отправить

<input type="submit"> определяет кнопку для отправки данных формы в обработчик формы .

<form action="/action_page.php">
  <label for="fname">First name:</label><br>
  <input type="text" id="fname" name="fname" value="John"><br>
  <label for="lname">Last name:</label><br>
  <input type="text" id="lname" name="lname" value="Doe"><br><br>
  <input type="submit" value="Отправить">
</form>


Тип ввода Сброс

<input type="reset"> определяет кнопку сброса , которая сбрасывает все значения формы к значениям по умолчанию:

<form action="/action_page.php">
  <label for="fname">First name:</label><br>
  <input type="text" id="fname" name="fname" value="John"><br>
  <label for="lname">Last name:</label><br>
  <input type="text" id="lname" name="lname" value="Doe"><br><br>
  <input type="submit" value="Отправить">
  <input type="reset" value="Reset">
</form>

Тип ввода Радио

<input type="radio"> определяет переключатель .

<p>Choose your favorite Web language:</p>

<form>
  <input type="radio" id="html" name="fav_language" value="HTML">
  <label for="html">HTML</label><br>
  <input type="radio" id="css" name="fav_language" value="CSS">
  <label for="css">CSS</label><br>
  <input type="radio" id="javascript" name="fav_language" value="JavaScript">
  <label for="javascript">JavaScript</label>
</form>

Тип ввода Флажок

Флажки позволяют пользователю выбирать ОДИН или БОЛЬШЕ вариантов из ограниченного числа вариантов.

<form>
  <input type="checkbox" id="vehicle1" name="vehicle1" value="Bike">
  <label for="vehicle1"> I have a bike</label><br>
  <input type="checkbox" id="vehicle2" name="vehicle2" value="Car">
  <label for="vehicle2"> I have a car</label><br>
  <input type="checkbox" id="vehicle3" name="vehicle3" value="Boat">
  <label for="vehicle3"> I have a boat</label>
</form>

<form>
  <label for="favcolor">Select your favorite color:</label>
  <input type="color" id="favcolor" name="favcolor">
</form>

Тип ввода Дата

Используется <input type="date"> для полей ввода, которые должны содержать дату.

<form>

</form>

Тип ввода Месяц

Позволяет <input type="month"> пользователю выбрать месяц и год.

<form>
  <label for="bdaymonth">Birthday (month and year):</label>
  <input type="month" id="bdaymonth" name="bdaymonth">

Тип ввода числа с ограничении

Определяет <input type="number"> поле числового ввода.

<form>
  <label for="quantity">Введите (от 1 до 5):</label>
  <input type="number" id="quantity" name="quantity" min="1" max="5">
</form>

Ограничения ввода

Вот список некоторых распространенных ограничений ввода:

Атрибут Значение
checked Указывает, что поле ввода должно быть предварительно выбрано при загрузке страницы (для type="checkbox" или type="radio")
disabled Указывает, что поле ввода должно быть отключено
max Указание максимального значения для поля ввода
maxlength Указание максимального количества символов для поля ввода
min Указание минимального значения для поля ввода
pattern Указание регулярного выражения для проверки входного значения на соответствие
readonly Указывает, что поле ввода предназначено только для чтения (не может быть изменено)
required Указывает, что поле ввода является обязательным (должно быть заполнено)
size Задание ширины (в символах) поля ввода
step Определяет допустимые интервалы чисел для поля ввода
value Определяет значение по умолчанию для поля ввода
<form>
  <label for="quantity">Quantity:</label>
  <input type="number" id="quantity" name="quantity" min="0" max="100" step="10" value="30">
</form>

Диапазон типов ввода

Определяет <input type="range"> элемент управления для ввода числа, точное значение которого не имеет значения (например, ползунок). Диапазон по умолчанию — от 0 до 100. Однако вы можете установить ограничения на то, какие числа принимаются min , с помощью атрибутов max , и : step

<form>
  <label for="vol">Volume (between 0 and 50):</label>
  <input type="range" id="vol" name="vol" min="0" max="50">
</form>

Поиск типа ввода

Используется <input type="search"> для полей поиска (поле поиска ведет себя как обычное текстовое поле).

<form>
  <label for="gsearch">Search Google:</label>
  <input type="search" id="gsearch" name="gsearch">
</form>

Тип входа Тел.

Используется <input type="tel"> для полей ввода, которые должны содержать номер телефона.

<form>
  <label for="phone">Enter your phone number:</label>
  <input type="tel" id="phone" name="phone" pattern="[0-9]{3}-[0-9]{2}-[0-9]{3}">
</form>

Тип входа Время

Позволяет <input type="time"> пользователю выбрать время (без часового пояса).

<form>
  <label for="appt">Select a time:</label>
  <input type="time" id="appt" name="appt">
</form>

<form>
  <label for="homepage">Введите вашу ссылку:</label>
