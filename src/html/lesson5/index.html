HTML-ссылки — целевой атрибут

        По умолчанию связанная страница будет отображаться в текущем окне браузера. Чтобы изменить это, необходимо указать другую цель для ссылки.

        Атрибут targetуказывает, где открыть связанный документ.

        Атрибут targetможет иметь одно из следующих значений:

        _self- По умолчанию. Открывает документ в том же окне/вкладке, в котором он был щелкнут.
        _blank- Открывает документ в новом окне или вкладке.
        _parent- Открывает документ в родительском фрейме
        _top- Открывает документ во всем окне.

<a href="#" target="_blank">Ссылка на страницу</a>

Абсолютные URL-адреса и относительные URL-адреса

<h2>Absolute URLs</h2>
<p><a href="https://www.w3.org/">W3C</a></p>
<p><a href="https://www.google.com/">Google</a></p>

<h2>Relative URLs</h2>
<p><a href="html_images.asp">HTML Images</a></p>
<p><a href="/css/default.asp">CSS Уроки</a></p>

HTML-ссылки — используйте изображение в качестве ссылки

<a href="default.asp">
<img src="smiley.gif" alt="HTML Уроки" style="width:42px;height:42px;">
</a>


Ссылка на адрес электронной почты

<a href="mailto:someone@example.com">Отправить email</a>


Кнопка как ссылка

<button onclick="document.location='default.asp'">HTML Уроки</button>

Заголовки ссылок

<a href="#" title="Загаловок ссылки">Ссылка на HTML Уроки</a>

Краткое содержание урока

  • Используйте <a> элемент для определения ссылки
  • Используйте href атрибут, чтобы определить адрес ссылки.
  • Используйте target атрибут, чтобы определить, где открыть связанный документ.
  • Используйте <img> элемент (внутри <a> ), чтобы использовать изображение в качестве ссылки.
  • Используйте mailto: схему внутри href атрибута, чтобы создать ссылку, открывающую почтовую программу пользователя.

Домашнее задание: Работа с ссылками и изображениями

  1. Создайте HTML-документ, который содержит следующие ссылки и элементы:

    • Обычную ссылку <a> с текстом "Ссылка на страницу", ведущую на пустую страницу (используйте href="#").
    • Ссылку, которая открывает новое окно или вкладку браузера, с текстом "Ссылка на страницу" и атрибутом target="_blank".
    • Две абсолютные ссылки на страницы "W3C" и "Google".
    • Две относительные ссылки на страницы "HTML Images" и "CSS Уроки".
    • Используйте изображение в качестве ссылки, где изображение "smiley.gif" работает как ссылка на "HTML Уроки".

  2. Создайте ссылку на адрес электронной почты с текстом "Отправить email" и атрибутом href="mailto:someone@example.com".

  3. Преобразуйте кнопку в ссылку, где при клике на кнопку "HTML Уроки" страница перенаправляется на "default.asp".

  4. Добавьте заголовок к ссылке с текстом "Ссылка на HTML Уроки" и атрибутом title="Заголовок ссылки".
