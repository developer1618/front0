Синтаксис HTML-изображений

            Тег HTML используется для вставки изображения на веб-страницу.

    Изображения технически не вставляются на веб-страницу; изображения связаны с веб-страницами. Тег создает пространство хранения для изображения, на которое указывает ссылка.

    Тег пуст, содержит только атрибуты и не имеет закрывающего тега.

    Тег имеет два обязательных атрибута:

    src — указывает путь к изображению
    alt — указывает альтернативный текст для изображения.


Атрибут src

<img src="img_chania.jpg" alt="Flowers in Chania">


Атрибут alt

<img src="img_chania.jpg" alt="Flowers in Chania">


Размер изображения — ширина и высота

<img src="img_girl.jpg" alt="Girl in a jacket" style="width:500px;height:600px;">

<img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">

HTML- фоновые изображения

Чтобы добавить фоновое изображение в элемент HTML, используйте styleатрибут HTML и свойство CSS background-image:

<p style="background-image: url('img_girl.jpg');">

Также вы можете указать фоновое изображение в <style> элементе, в <head> разделе:


<style>
p {
  background-image: url('img_girl.jpg');
}
</style>

Чтобы фоновое изображение не повторялось, установите background-repeat для свойства значение no-repeat.


<style>
body {
  background-image: url('example_img_girl.jpg');
  background-repeat: no-repeat;
}
</style>

HTML - элемент <picture>

Элемент HTML <picture> позволяет отображать разные изображения для разных устройств или размеров экрана.

<picture>
  <source media="(min-width: 650px)" srcset="img_food.jpg">
  <source media="(min-width: 465px)" srcset="img_car.jpg">
  <img src="img_girl.jpg">
</picture>

Краткое содержание урока

        Используйте элемент HTML <img> для определения изображения
        Используйте атрибут HTML src, чтобы определить URL-адрес изображения.
        Используйте атрибут HTML alt, чтобы определить альтернативный текст для изображения, если его невозможно отобразить.
        Используйте HTML widthи heightатрибуты или CSS width и height свойства, чтобы определить размер изображения.
        Используйте свойство CSS float, чтобы позволить изображению перемещаться влево или вправо.

Домашнее задание: Работа с изображениями в HTML

  1. Создайте HTML-документ и добавьте в него изображение с использованием элемента <img>. Укажите путь к изображению в атрибуте src и добавьте альтернативный текст с помощью атрибута alt.

  2. Установите размер изображения, используя атрибуты width и height. Попробуйте также установить размер изображения с использованием стилей CSS, например, с помощью свойств style="width:500px;height:600px;".

  3. Создайте анимированное изображение и добавьте его в HTML. Укажите размер и альтернативный текст для этого изображения.

  4. Преобразуйте изображение в ссылку. Оберните изображение в элемент <a>, который будет перенаправлять пользователя на какую-либо страницу.

  5. Создайте плавающее изображение, используя свойство CSS float. Попробуйте выровнять изображение как справа, так и слева от текста.

  6. Добавьте фоновое изображение к элементу HTML с использованием стиля CSS. Попробуйте установить фоновое изображение, чтобы оно не повторялось на странице.

  7. Используйте элемент <picture> для отображения разных изображений в зависимости от размеров экрана устройства. Создайте <picture> с разными <source> элементами и указанными изображениями.
