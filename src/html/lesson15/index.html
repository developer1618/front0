Семантические элементы HTML

Семантический элемент четко описывает свое значение как браузеру, так и разработчику.
В HTML есть несколько семантических элементов, которые можно использовать для определения различных частей веб-страницы:

Тег Описание
<article> Определяет независимый, автономный контент
<aside> Определяет содержимое помимо содержимого страницы
<details> Определяет дополнительные сведения, которые пользователь может просмотреть или скрыть
<figcaption> Определяет подпись для <figure> элемент
<figure> Определяет автономный контент, например иллюстрации, диаграммы, фотографии, код. объявления и т. д.
<footer> Определяет нижний колонтитул документа или раздела
<header> Указывает заголовок документа или раздела
<main> Указывает основное содержимое документа
<mark> Определяет отмеченный/выделенный текст
<nav> Определяет навигационные ссылки
<section> Определяет раздел в документе
<summary> Определяет видимый заголовок для <details> элемент
<time> Определяет дату/время

HTML-элемент <section>

Элемент <section> определяет раздел в документе.

Примеры <section> использования элемента:



  • Главы
  • Введение
  • News items
  • Контактная информация

<section>
<h1>WWF</h1>
<p>Всемирный фонд природы (WWF) – это международная организация, занимающаяся вопросами сохранения, исследование и восстановление окружающей среды, ранее называвшейся Миром Фонд дикой природы. WWF был основан в 1961 году.</p>
</section>

<section>
<h1>Символ панды WWF</h1>
<p>Панда стала символом WWF. Известный логотип WWF в виде панды произошел от панды по имени Чи Чи, которая был переведен из Пекинского зоопарка в Лондонский зоопарк в том же году. создание WWF.</p>
</section>

HTML-элемент <article>

Этот <article> элемент определяет независимый, автономный контент.

Примеры <article> использования элемента:

  • Сообщения на форуме
  • Сообщения в блоге
  • Комментарии пользователей
  • Карточки товаров
  • Газетные статьи

<article>
<h2>Google Chrome</h2>
<p>Google Chrome — веб-браузер, разработанный компанией Google и выпущенный в 2008 году. Сегодня Chrome — самый популярный веб-браузер в мире!</p>
</article>

<article>
<h2>Mozilla Firefox</h2>
<p>Mozilla Firefox — веб-браузер с открытым исходным кодом, разработанный Mozilla. Firefox является вторым по популярности веб-браузером с января 2018 года.</p>
</article>

<article>
<h2>Microsoft Edge</h2>
<p>Microsoft Edge — веб-браузер, разработанный Microsoft, выпущен в 2015 году. Microsoft Edge заменил Internet Explorer.</p>
</article>

HTML-элемент <header>

Элемент <header> представляет собой контейнер для вводного контента или набор навигационных ссылок.

Элемент <header> обычно содержит:

  • один или несколько элементов заголовка (<h1> – <h6>)
  • логотип или значок
  • информация об авторстве


HTML-элемент <footer>

Элемент <footer> определяет нижний колонтитул документа или раздела.

Элемент <footer> обычно содержит:

  • информация об авторстве
  • информация об авторских правах
  • Контактная информация
  • карта сайта
  • вернуться к началу ссылок
  • связанные документы
<footer>
  <p>Автор: Админ Админов</p>
  <p><a href="mailto:info@admin.tj">info@admin.tj</a></p>
</footer>


HTML-элемент <nav>

Элемент <nav> определяет набор навигационных ссылок.

<nav>
  <a href="/html/">HTML</a> |
  <a href="/css/">CSS</a> |
  <a href="/js/">JavaScript</a> |
  <a href="/jquery/">jQuery</a>
</nav>

HTML-элемент <aside>

Элемент <aside> определяет некоторый контент помимо контента, в котором он находится (например, боковая панель).

<p>Этим летом мы с семьей посетили центр Epcot. Погода была хорошая, и Эпкот был потрясающим! Я отлично провел лето вместе со своей семьей!</p>

<aside>
<h4>Эпкот Центр</h4>
<p>Эпкот — это тема парк на курорте Walt Disney World Resort с захватывающими аттракционами, международные павильоны, отмеченные наградами фейерверки и сезонные специальные предложения события.</p>
</aside>

Домашнее задание: Использование семантических элементов HTML

Создайте HTML-страницу, используя семантические элементы HTML5 для следующих разделов:

  1. Заголовок страницы, содержащий название вашей страницы.

  2. Навигационное меню с ссылками на разные разделы страницы.

  3. Главное содержимое страницы, включая заголовок и несколько абзацев текста о выбранной вами теме.

  4. Боковая панель (element <aside>), содержащая дополнительную информацию или связанные ссылки.

  5. Подвал страницы (element <footer>), включающий информацию об авторстве, контактные данные и ссылки на социальные сети (если у вас есть такие).

  6. Создайте стили CSS, чтобы придать вашей странице некоторый визуальный стиль. Вы можете добавить цвет фона, шрифты, отступы и рамки.

  7. Добавьте содержание в каждый раздел, чтобы ваша страница выглядела и звучала как что-то логичное и полезное. Это может быть что угодно, от статей о путешествиях до личных блогов или портфолио.

  8. Убедитесь, что ваша страница адаптивна для разных размеров экрана, используя медиа-запросы.

  9. После создания страницы, опубликуйте ее в интернете, чтобы вы могли поделиться ей с друзьями или преподавателями.
