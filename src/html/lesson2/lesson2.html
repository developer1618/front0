<h1>Это загаловок</h1>

</body>

<body>

<h1 style="background-color:powderblue;">Это загаловок</h1>
<p style="background-color:tomato;">Это параграф</p>

</body>


Цвет текста

<h1 style="color:blue;">This is a heading</h1>
<p style="color:red;">This is a paragraph.</p>


Шрифты

<h1 style="font-family:verdana;">This is a heading</h1>
<p style="font-family:courier;">This is a paragraph.</p>


Размер текста

<h1 style="font-size:300%;">This is a heading</h1>
<p style="font-size:160%;">This is a paragraph.</p>


Выравнивание текста

<h1 style="text-align:center;">Centered Heading</h1>
<p style="text-align:center;">Centered paragraph.</p>

Краткое содержание урока

    Используйте style атрибут для стилизации HTML-элементов.
    Использовать background-color для цвета фона
    Использовать color для цвета текста
    Использовать font-family для текстовых шрифтов
    Использовать font-size для размеров текста
    Использовать text-align для выравнивания текста
