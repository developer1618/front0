HTML -фреймы Iframe

HTML iframe используется для отображения веб-страницы внутри веб-страницы.

<iframe src="url" title="description"></iframe>

Iframe – установка высоты и ширины

Используйте атрибуты heightи width, чтобы указать размер iframe.
По умолчанию высота и ширина указаны в пикселях:

<iframe src="demo_iframe.htm" height="200" width="300" title="Iframe Example"></iframe>

<iframe src="demo_iframe.htm" style="height:200px;width:300px;" title="Iframe Example"></iframe>


Iframe — удалить границу

<iframe src="demo_iframe.htm" style="border:none;" title="Iframe Example"></iframe>


Iframe — цель для ссылки

iframe можно использовать в качестве целевого фрейма для ссылки.
Атрибут target ссылки должен ссылаться на name атрибут iframe:

<iframe src="demo_iframe.htm" name="iframe_a" title="Iframe Example"></iframe>

<p><a href="https://www.w3schools.com" target="iframe_a">W3Schools.com</a></p>


Домашнее задание: Работа с iframe в HTML

  1. Создайте HTML-страницу с использованием фрейма (iframe), который встраивает внутрь себя другую веб-страницу, например, сайт www.admin.tj. Установите размер фрейма, чтобы он отображал внутреннюю страницу в определенных размерах.

  2. Измените заголовок фрейма, используя атрибут title.

  3. На странице с фреймом, добавьте ссылку, которая перенаправляет пользователя на внешний сайт. При этом, целевой фрейм для загрузки внешнего сайта должен быть вашим фреймом.

  4. Добавьте на свою страницу кнопку, и сделайте так, чтобы при нажатии на нее загружалась другая веб-страница внутри вашего фрейма.

  5. Стилизуйте ваш фрейм с помощью CSS: измените цвет фона, шрифты или другие стили по вашему усмотрению.

  6. Попробуйте удалить границу фрейма и поэкспериментируйте с другими стилями.

  7. Внимательно прокомментируйте ваш код, объясняя, что делает каждая часть.

  8. Добавьте описание к вашей HTML-странице и украсьте ее, чтобы она выглядела привлекательной.

После выполнения этого задания, ученики будут знать, как вставлять и стилизовать фреймы в HTML, а также как создавать ссылки и кнопки для управления содержимым фрейма.
