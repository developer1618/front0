HTML-атрибуты

Атрибут href

Атрибут src
<img src=""> 

        Есть два способа указать URL-адрес в src атрибуте:
    
        1. Абсолютный URL-адрес — ссылки на внешнее изображение, размещенное на другом веб-сайте. Пример: src="https://www.webacademy.tj/img/image.jpg" .
    
        2. Относительный URL-адрес — ссылки на изображение, размещенное на веб-сайте. 
        Здесь URL-адрес не включает имя домена. Если URL-адрес начинается без косой черты, он будет относиться к текущей странице. 
        Пример: src="img_girl.jpg". Если URL-адрес начинается с косой черты, он будет относиться к домену. Пример: src="/images/img_girl.jpg".
    
        Совет: почти всегда лучше использовать относительные URL-адреса. Они не сломаются, если вы смените домен.

Одинарные или двойные кавычки?

Двойные кавычки вокруг значений атрибутов наиболее распространены в HTML, но можно использовать и одинарные кавычки.

<p title='John "ShotGun" Nelson'>

Краткое содержание урока

            Все элементы HTML могут иметь атрибуты.
            Атрибут href указывает <a> URL-адрес страницы, на которую ведет ссылка.
            Атрибут src указывает путь к изображению, которое будет отображаться. <img>
            Атрибуты width и height предоставляют <img> информацию о размере изображений.
            Атрибут alt предоставляет альтернативный текст для изображения  <img>
            Атрибут style используется для добавления стилей к элементу, таких как цвет, шрифт, размер и т. д.
            Атрибут lang тега <html> объявляет язык веб-страницы.
            Атрибут title определяет некоторую дополнительную информацию об элементе.
