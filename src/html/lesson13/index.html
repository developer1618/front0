Пути к файлам HTML

Путь к файлу описывает расположение файла в структуре папок веб-сайта.

Путь Описание
<img src="picture.jpg"> Файл «picture.jpg» находится в той же папке, что и текущая страница.
<img src="images/picture.jpg"> Файл «picture.jpg» находится в папке images в текущей папке.
<img src="/images/picture.jpg"> Файл «picture.jpg» находится в папке изображений в корне текущего веб-сайта.
<img src="../picture.jpg"> Файл «picture.jpg» находится в папке на один уровень выше текущей папки.

HTML — элементи шапки веб-сайта

Элемент HTML <head> является контейнером для следующих элементов: <title> , <style> , <meta> , <link> , <script> и <base> .

HTML-элемент <head>


Элемент <head> представляет собой контейнер для метаданных (данных о данных) и размещается между <html> тегом и <body> тегом.



HTML-элемент <title>

Элемент <title> определяет заголовок документа. Заголовок должен быть только текстовым и отображаться в строке заголовка браузера или на вкладке страницы.


<!DOCTYPE html>
<html>
<head>
  <title>A Meaningful Page Title</title>
</head>
<body>

The content of the document......

</body>
</html>

HTML-элемент <style>

Этот <style> элемент используется для определения информации о стиле для одной HTML-страницы:

<style>
  body {background-color: powderblue;}
  h1 {color: red;}
  p {color: blue;}
</style>

HTML-элемент <link>

Этот <link> элемент определяет связь между текущим документом и внешним ресурсом.

Тег <link> чаще всего используется для ссылки на внешние таблицы стилей:


<link rel="stylesheet" href="mystyle.css">

HTML-элемент <meta>

Этот <meta> элемент обычно используется для указания набора символов, описания страницы, ключевых слов, автора документа и настроек области просмотра.

Определите используемый набор символов:

<meta charset="UTF-8">

Определите ключевые слова для поисковых систем:

<meta name="keywords" content="HTML, CSS, JavaScript">

Определите описание вашей веб-страницы:

<meta name="description" content="Free Web tutorials">

Пример

<meta charset="UTF-8">
<meta name="description" content="Free Web tutorials">
<meta name="keywords" content="HTML, CSS, JavaScript">
<meta name="author" content="John Doe">

HTML-элемент <script>

Этот <script> элемент используется для определения JavaScript-кодов на стороне клиента.

<script>
function myFunction() {
  document.getElementById("demo").innerHTML = "Hello JavaScript!";
}
</script>


Краткое содержание урока

  • Элемент <head> представляет собой контейнер для метаданных (данных о данных).
  • Элемент <head> помещается между <html> тегом и <body> тегом
  • Элемент <title> является обязательным и определяет заголовок документа.
  • Этот <style> элемент используется для определения информации о стиле для одного документа.
  • Тег <link> чаще всего используется для ссылки на внешние таблицы стилей.
  • Этот <meta> элемент обычно используется для указания набора символов, описания страницы, ключевых слов, автора документа и настроек области просмотра.
  • Этот <script> элемент используется для определения JavaScript на стороне клиента.
  • Элемент <base> определяет базовый URL-адрес и/или цель для всех относительных URL-адресов на странице.

Домашнее задание: Работа с элементами <head> в HTML

  1. Создайте новый HTML-документ.

  2. Внутри <head> добавьте элемент <title> и установите в нем заголовок вашей страницы. Например: "Моя Заголовочная Страница".

  3. Добавьте элемент <style> в <head> и определите в нем некоторые стили (например, измените цвет фона и размер текста). Примените эти стили к какому-нибудь элементу внутри <body>.

  4. Используя элемент <link>, подключите внешний файл стилей (CSS файл) к вашей странице. Создайте этот CSS файл и определите в нем дополнительные стили. Убедитесь, что эти стили также применяются к элементам в вашем HTML-документе.

  5. Добавьте несколько элементов <meta> для указания набора символов (UTF-8), описания вашей страницы, ключевых слов и автора.

  6. Внутри <body> создайте какие-то элементы (например, заголовки, параграфы, изображения) и примените JavaScript с использованием <script> элемента для изменения их содержимого или стилей.

  7. Закомментируйте каждый элемент в вашем коде, чтобы пояснить, что он делает.

  8. Проверьте вашу страницу в браузере, убедитесь, что заголовок, стили и JavaScript работают правильно.

  9. Сохраните ваш файл HTML и все внешние файлы, которые вы создали (CSS файл и, если есть, изображения), в одну папку. Проверьте, что все ссылки на файлы в вашем HTML-документе корректны.

  10. Если возможно, поделитесь вашей работой с преподавателем или коллегами для обратной связи.
