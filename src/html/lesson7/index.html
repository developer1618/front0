HTML -фавикон

Фавиконка — это небольшое изображение, отображаемое рядом с заголовком страницы на вкладке браузера.


<!DOCTYPE html>
<html>
<head>
  <title>My Page Title</title>
  <link rel="icon" type="image/x-icon" href="/images/favicon.ico">
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>

</body>
</html>


HTML- заголовок страницы

Каждая веб-страница должна иметь заголовок, описывающий смысл страницы. Элемент <title>добавляет заголовок на вашу страницу:

<!DOCTYPE html>
<html>
<head>
  <title>HTML Tutorial</title>
</head>
<body>

The content of the document......

</body>
</html>

HTML- таблица Colspan и Rowspan

<table>
  <tr>
    <th colspan="2">Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Jill</td>
    <td>Smith</td>
    <td>43</td>
  </tr>
  <tr>
    <td>Eve</td>
    <td>Jackson</td>
    <td>57</td>
  </tr>
</table>

<table>
  <tr>
    <th>Name</th>
    <td>Jill</td>
  </tr>
  <tr>
    <th rowspan="2">Phone</th>
    <td>555-1234</td>
  </tr>
  <tr>
    <td>555-8745</td>
</tr>
</table>



Домашнее задание: Фавиконки и таблицы

  1. Добавьте фавиконку к вашей веб-странице. Создайте изображение фавиконки и используйте элемент <link> с атрибутами rel="icon" и type="image/x-icon" для связи фавиконки с вашей страницей.

  2. Определите заголовок вашей веб-страницы, используя элемент <title> в разделе <head> документа. Убедитесь, что заголовок описывает содержание страницы.

  3. Создайте HTML-таблицу, представляющую информацию о компаниях. Включите в таблицу следующие данные: название компании, контактное лицо и страну. Используйте теги <table>, <tr>, <th>, и <td>.

  4. Задайте пунктирные границы для ячеек вашей таблицы, используя свойство border-style для элементов <table> и <td>. Попробуйте разные стили границ, такие как dotted, dashed, и double.

  5. В вашей таблице используйте атрибуты colspan и rowspan для объединения ячеек. Создайте заголовок таблицы, который объединяет две ячейки, и также объедините ячейки для отображения информации о телефоне компании.

  6. Добавьте еще несколько строк и компаний в вашу таблицу, чтобы продемонстрировать ее универсальность и разнообразие.
