HTML- списки

Списки HTML позволяют веб-разработчикам группировать набор связанных элементов в списки.

Неупорядоченный список HTML:

  • Элемент
  • Элемент
  • Элемент
  • Элемент

Упорядоченный список HTML:

  1. Первый предмет
  2. Второй предмет
  3. Третий пункт
  4. Четвертый пункт

<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>

<ol>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>


Неупорядоченный список HTML — выбор маркера элемента списка

Пример — диск

<ul style="list-style-type:disc;">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>

Пример — круг

<ul style="list-style-type:circle;">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>

Пример — квадрат

<ul style="list-style-type:square;">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>

Горизонтальный список с помощью CSS

Списки HTML можно стилизовать разными способами с помощью CSS.
Один из популярных способов — горизонтальное оформление списка для создания меню навигации:

<!DOCTYPE html>
<html>
<head>
<style>
ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
  overflow: hidden;
  background-color: #333333;
}

li {
  float: left;
}

li a {
  display: block;
  color: white;
  text-align: center;
  padding: 16px;
  text-decoration: none;
}

li a:hover {
  background-color: #111111;
}
</style>
</head>
<body>

<ul>
  <li><a href="#home">Home</a></li>
  <li><a href="#news">News</a></li>
  <li><a href="#contact">Contact</a></li>
  <li><a href="#about">About</a></li>
</ul>

</body>
</html>


Упорядоченный список HTML

Числа:

<ol type="1">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

Заглавные буквы:

<ol type="A">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

Строчные буквы:

<ol type="a">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

Прописные римские цифры:

<ol type="I">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

Строчные римские цифры:

<ol type="i">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

Краткое содержание урока

            Используйте элемент HTML <ul> для определения неупорядоченного списка.
            Используйте свойство CSS list-style-typeдля определения маркера элемента списка.
            Используйте элемент HTML <li> для определения элемента списка
            Списки могут быть вложенными
            Элементы списка могут содержать другие элементы HTML.
            Используйте свойство CSS float:left для отображения списка по горизонтали.

Домашнее задание: Работа с списками в HTML

  1. Создайте неупорядоченный список <ul>, в котором будет несколько элементов <li>, представляющих ваши любимые фрукты. Добавьте стилизацию для маркеров списка, чтобы они отображались как круги, квадраты и диски.

  2. Создайте упорядоченный список <ol>, который будет содержать ваши любимые фильмы. Варьируйте вид маркера списка, используя разные типы (цифры, заглавные и строчные буквы, римские цифры).

  3. Создайте вложенные списки. Внутри элементов списка из задания 1 добавьте еще один неупорядоченный список с дополнительными элементами.

  4. Создайте горизонтальное меню навигации с использованием списка. Используйте CSS для стилизации списка и его элементов, чтобы создать горизонтальное меню. Добавьте несколько пунктов меню и при наведении курсора на пункт меню меняйте его цвет фона.

  5. Используя элементы списка, создайте структурированный список с пунктами "Введение", "Раздел 1", "Раздел 2" и "Заключение". Вложите в "Раздел 1" и "Раздел 2" дополнительные элементы списка.

  6. Создайте дополнительные списки с разными видами маркеров, такими как квадраты и диски. Выведите в них любую информацию на ваш выбор.
