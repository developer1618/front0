Элементы и методы HTML- макета

Веб-сайты часто отображают контент в нескольких столбцах.


Элементы HTML-макета

  • <header> - Определяет заголовок документа или раздела.
  • <nav> - Определяет набор навигационных ссылок
  • <section> - Определяет раздел в документе.
  • <article> - Определяет независимый, самодостаточный контент
  • <aside> - Определяет контент отдельно от контента (например, боковая панель).
  • <footer> - Определяет нижний колонтитул документа или раздела.
  • <details> - Определяет дополнительные детали, которые пользователь может открывать и закрывать по требованию.
  • <summary> - Определяет заголовок для <details> элемента



HTML- адаптивный веб-дизайн

Адаптивный веб-дизайн — это создание веб-страниц, которые хорошо выглядят на всех устройствах!
Адаптивный веб-дизайн автоматически подстраивается под различные размеры экрана и области просмотра.



Настройка области просмотра

<meta> Чтобы создать адаптивный веб-сайт, добавьте на все свои веб-страницы следующий тег:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Адаптивные изображения

<img src="img_girl.jpg" style="width:100%;">

Использование свойства max-width

<img src="img_girl.jpg" style="max-width:100%;height:auto;">

Показывать разные изображения в зависимости от ширины браузера

<picture>
  <source srcset="img_smallflower.jpg" media="(max-width: 600px)">
  <source srcset="img_flowers.jpg" media="(max-width: 1500px)">
  <source srcset="flowers.jpg">
  <img src="img_smallflower.jpg" alt="Flowers">
</picture>

Адаптивный размер текста

<h1 style="font-size:10vw">Hello World</h1>

Медиа-запросы

Помимо изменения размера текста и изображений, на адаптивных веб-страницах также часто используются медиа-запросы.

<style>
.left, .right {
  float: left;
  width: 20%; /* The width is 20%, by default */
}

.main {
  float: left;
  width: 60%; /* The width is 60%, by default */
}

/* Use a media query to add a breakpoint at 800px: */
@media screen and (max-width: 800px) {
  .left, .main, .right {
    width: 100%; /* The width is 100%, when the viewport is 800px or smaller */
  }
}
</style>



Домашнее задание: Создание адаптивной веб-страницы

  1. Создайте новый HTML-документ.

  2. В <head> вашего HTML-документа добавьте следующий код для настройки области просмотра для адаптивного веб-дизайна:

html
<meta name="viewport" content="width=device-width, initial-scale=1.0">

  1. Создайте адаптивную веб-страницу, используя семантические элементы HTML5, такие как <header>, <nav>, <section>, <article>, <aside>, <footer>, <details>, и <summary>. Задайте структуру страницы с использованием этих элементов.

  2. Вставьте несколько изображений на вашу страницу и настройте их размер так, чтобы они адаптировались к различным размерам экрана. Используйте свойства width:100%; и max-width:100%;height:auto; для изображений.

  3. Используйте медиа-запросы и CSS для создания адаптивной макета. Создайте стили, которые будут работать на разных размерах экрана. Например, создайте стили для разных разрешений экрана или разных устройств (смартфоны, планшеты, десктопы).

  4. Сделайте вашу веб-страницу адаптивной, так, чтобы она хорошо выглядела как на маленьких экранах, так и на больших.

  5. Проверьте вашу веб-страницу в различных браузерах и на разных устройствах, чтобы убедиться, что она выглядит хорошо и работает правильно.

  6. Поделитесь вашей работой с кем-то из своих коллег или преподавателем и попросите обратную связь.
