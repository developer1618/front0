HTML - Урок 12

HTML JavaScript

JavaScript делает HTML-страницы более динамичными и интерактивными.

Первый JavaScript код


HTML-тег <script>

Тег HTML <script> используется для определения сценария на стороне клиента (JavaScript).

<script>
document.getElementById("demo").innerHTML = "Hello JavaScript!";
</script>


Немного JavaScript

JavaScript может изменять содержимое:

document.getElementById("demo").innerHTML = "Hello JavaScript!";

JavaScript может менять стили:

document.getElementById("demo").style.fontSize = "25px";
document.getElementById("demo").style.color = "red";
document.getElementById("demo").style.backgroundColor = "yellow";

JavaScript может изменять атрибуты:

document.getElementById("image").src = "picture.gif";

HTML-тег <noscript>

Тег HTML <noscript> определяет альтернативный контент, который будет отображаться пользователям, которые отключили сценарии в своем браузере или имеют браузер, который не поддерживает сценарии:

<script>
document.getElementById("demo").innerHTML = "Hello JavaScript!";
</script>
<noscript>Sorry, your browser does not support JavaScript!</noscript>



Домашнее задание: Основы JavaScript

  1. Создайте HTML-страницу, включив на ней блок <div> с id="demo" и элемент <img> с id="image". Эти элементы будут использоваться для выполнения заданий.

  2. Используя JavaScript, измените содержимое блока с id="demo" на "Hello, JavaScript!".

  3. Измените стили блока с id="demo", чтобы установить его размер шрифта равным 25 пикселям, цвет шрифта на красный и цвет фона на желтый.

  4. Измените атрибут src элемента <img> с id="image" на "picture.gif" с использованием JavaScript.

  5. Вставьте тег <noscript> после JavaScript-кода в вашем HTML-документе. В этом теге укажите сообщение, которое будет отображаться пользователям, не поддерживающим JavaScript. Например, "Извините, ваш браузер не поддерживает JavaScript!".

  6. Добавьте кнопку или ссылку на вашей странице с надписью "Изменить стиль", и используйте JavaScript, чтобы при нажатии на эту кнопку изменить стиль элемента с id="demo" на свой выбор (например, изменить цвет фона на синий и размер шрифта на 30 пикселей).

  7. Внимательно прокомментируйте ваш код, объясняя, что делает каждая часть.

  8. Добавьте описание к вашей HTML-странице и украсьте ее, чтобы она выглядела привлекательной.
