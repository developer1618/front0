HTML5 SVG

SVG определяет векторную графику в формате XML.

Что такое SVG?

  • SVG расшифровывается как масштабируемая векторная графика
  • SVG используется для определения графики для bнтернета
  • SVG - это рекомендация W3C

Элемент <svg>

HTML элемент <svg> - это контейнер для SVG графики.

SVG имеет несколько методов рисования контуров, прямоугольников, кругов, текста и графических изображений.

SVG круг

<!DOCTYPE html>
<html>
 <body>

  <svg width="100" height="100">
   <circle cx="50" cy="50" r="40" stroke="green" stroke-width="4" fill="yellow" />
  </svg>

 </body>
</html>

SVG прямоугольник

<svg width="400" height="100">
  <rect width="400" height="100" style="fill:rgb(0,0,255);stroke-width:10;stroke:rgb(0,0,0)" />
</svg>

SVG Прямоугольник с закругленными углами

<svg width="400" height="180">
  <rect x="50" y="20" rx="20" ry="20" width="150" height="150"
  style="fill:red;stroke:black;stroke-width:5;opacity:0.5" />
</svg>

SVG звезда

<svg width="300" height="200">
  <polygon points="100,10 40,198 190,78 10,78 160,198"
  style="fill:lime;stroke:purple;stroke-width:5;fill-rule:evenodd;" />
</svg>

SVG логотип

<svg height="130" width="500">
 <defs>
   <linearGradient id="grad1" x1="0%" y1="0%" x2="100%" y2="0%">
     <stop offset="0%" style="stop-color:rgb(255,255,0);stop-opacity:1" />
     <stop offset="100%" style="stop-color:rgb(255,0,0);stop-opacity:1" />
   </linearGradient>
 </defs>
 <ellipse cx="100" cy="70" rx="85" ry="55" fill="url(#grad1)" />
 <text fill="#ffffff" font-size="45" font-family="Verdana" x="50" y="86">SVG</text>
 К сожалению, Ваш браузер не поддерживает встроенный SVG.
</svg>

Домашняя задания

Задача 1: Создание анимированного солнца Создайте SVG-графику с изображением солнца и добавьте анимацию так, чтобы солнце поднималось над горизонтом и садилось обратно. Используйте атрибуты <animate> для управления движением солнца.

Задача 2: Нарисовать простую иконку Используя SVG, нарисуйте простую иконку, например, иконку дома, сердца или звезды. Добавьте контур и заливку для иконки, чтобы сделать её красочной.

Задача 3: Создание фигурной композиции Создайте SVG-графику, представляющую абстрактную фигурную композицию из нескольких геометрических фигур, таких как круги, квадраты, и треугольники. Расположите фигуры на холсте так, чтобы они создали интересный дизайн.
