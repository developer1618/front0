JavaScript Циклы

Циклы удобны, если вы хотите запускать один и тот же код снова и снова, каждый раз с другим значением.

text += cars[0] + "<br>";
text += cars[1] + "<br>";
text += cars[2] + "<br>";
text += cars[3] + "<br>";
text += cars[4] + "<br>";
text += cars[5] + "<br>";

var i;
for (i = 0; i < cars.length; i++) {
  text += cars[i] + "<br>";
}

Цикл for

Цикл for имеет следующий синтаксис:

for (i = 0; i < 5; i++) {
  text += "Число является " + i + "<br>";
}

Цикл for/in

JavaScript оператор for/in перебирает свойства объекта:

var person = {fname:"Щипунов", lname:"Андрей", age:25};

var text = "";
var x;
for (x in person) {
  text += person[x];
}

Цикл for/of

JavaScript оператор for/of перебирает значения итерируемых объектов.
Цикл for/of позволяет перебирать итерируемые структуры данных, такие как массивы, строки, карты, списки узлов Node и т.д.

var cars = ["БМВ", "Вольво", "Мини"];
var x;

for (x of cars) {
  document.write(x + "<br >");
}

var txt = "JavaScript";
var x;

for (x of txt) {
  document.write(x + "<br >");
}

JavaScript Цикл For In

JavaScript оператор for in перебирает свойства объекта:

const person = {fname:"Щипунов", lname:"Андрей", age:25};

let text = "";
for (let x in person) {
text += person[x];
}
  • Цикл for in выполняет итерацию по объекту person
  • Каждая итерация возвращает ключ (x)
  • Ключ используется для доступа к значению ключа
  • Значение ключа равно person[x]

Метод For In над массивами

JavaScript оператор for in, также может перебирать свойства массива:

const numbers = [45, 4, 9, 16, 25];

let txt = "";
for (let x in numbers) {
txt += numbers[x];
}

Метод массива forEach()

Метод forEach() вызывает функцию (функцию обратного вызова) один раз для каждого элемента массива.

const numbers = [45, 4, 9, 16, 25];

let txt = "";
numbers.forEach(myFunction);

function myFunction(value, index, array) {
txt += value;
}

JavaScript Цикл For Of

JavaScript оператор for of перебирает значения итеративного объекта.

const cars = ["БМВ", "Вольво", "Мини"];

let text = "";
for (let x of cars) {
text += x;
}

let language = "JavaScript";

let text = "";
for (let x of language) {
text += x;

JavaScript Цикл While

Цикл while проходит через блок кода до тех пор, пока заданное условие истинно.

while (i < 10) {
  text += "Число является " + i;
  i++;
}

Цикл do/while

Цикл do/while является вариантом цикла while. Этот цикл будет выполнять блок кода один раз, прежде чем проверять, выполняется ли условие, а затем будет повторять цикл, пока условие выполняется.

do {
  text += "Число является " + i;
  i++;
}
while (i < 10);

Сравнение For и While

Если вы читали предыдущую главу о цикле for, вы могли заметить, что цикл while очень похож на цикл for, но оператор 1 и оператор 3 опущены.

var cars = ["БМВ", "Вольво", "Сааб", "Форд"];
var i = 0;
var text = "";

for (;cars[i];) {
  text += cars[i] + "<br>";
  i++;
}
