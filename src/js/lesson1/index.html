JavaScript Введение

Несколько примеров того, что может делать JavaScript.

JavaScript может изменять содержимое HTML

Одним из многих методов JavaScript HTML является getElementById().

Пример ниже "находит" элемент HTML (с id = "demo") и меняет его содержимое (innerHTML) на "Привет JavaScript":

document.getElementById("demo").innerHTML = "Привет JavaScript";

JavaScript может изменять значения атрибутов HTML

В этом примере JavaScript изменяет значение src атрибута (source) <img> тега:


JavaScript может изменять стили HTML (CSS)

Изменение стиля HTML элемента - это вариант изменения атрибута HTML:

document.getElementById("demo").style.fontSize = "35px";

JavaScript может скрывать элементы HTML

Скрыть элементы HTML можно, изменив display стиль:

document.getElementById("demo").style.display = "none";

JavaScript может отображать элементы HTML

Отображение скрытых HTML элементов также можно выполнить, изменив display стиль:

document.getElementById("demo").style.display = "block";

JavaScript Где установить?

Тег <script>

В HTML, JavaScript код вставляется между <script> и </script> теги.

<script>
document.getElementById("demo").innerHTML = "Мой Первый JavaScript";
</script>

JavaScript в <head>

В этом примере JavaScript function помещается в <head> раздел HTML страницы.

<!DOCTYPE html>
<html>

<head>
<script>
function myFunction() {
  document.getElementById("demo").innerHTML = "Параграф изменён.";
}
</script>
</head>
<body>

<h1>A Web Page</h1>
<p id="demo">A Paragraph</p>
<button type="button" onclick="myFunction()">Попробовать</button>

</body>
</html>


JavaScript в <body>

<!DOCTYPE html>
<html>
<body>

<h1>A Web Page</h1>
<p id="demo">A Paragraph</p>
<button type="button" onclick="myFunction()">Попробовать</button>

<script>
function myFunction() {
  document.getElementById("demo").innerHTML = "Параграф изменён.";
}
</script>

</body>
</html>

Внешний JavaScript

Скрипты также можно размещать во внешних файлах:

<script src="myScript.js"></script>

function myFunction() {
  document.getElementById("demo").innerHTML = "Параграф изменён.";
}

Преимущества внешнего JavaScript

Размещение скриптов во внешних файлах имеет ряд преимуществ:

  • Он разделяет HTML и код
  • Это упрощает чтение и поддержку HTML и JavaScript
  • Кэшированные файлы JavaScript могут ускорить загрузку страницы

Внешние ссылки

На внешние скрипты можно ссылаться с помощью полного URL-адреса или пути относительно текущей веб страницы.

<script src="https://www.admin.tj/js/myScript1.js"></script>

JavaScript Вывод

Возможности вывода JavaScript

JavaScript может "отображать" данные по-разному:

  • Запись в элемент HTML с использованием innerHTML.
  • Запись в вывод HTML с использованием document.write().
  • Запись в окно предупреждения с использованием window.alert().
  • Запись в консоль браузера с использованием console.log().

Использование innerHTML

Чтобы получить доступ к элементу HTML, JavaScript может использовать этот document.getElementById(id) метод.

Атрибут id определяет HTML элемент. Свойство innerHTML определяет содержание HTML:

<!DOCTYPE html>
<html>
<body>

<h1>Моя первая веб страница</h1>
<p>Мой первый параграф</p>

<p id="demo"></p>

<script>
document.getElementById("demo").innerHTML = 5 + 6;
</script>

</body>
</html>

Использование document.write()

Для тестирования удобно использовать document.write():

<!DOCTYPE html>
<html>
<body>

<h1>Моя первая веб страница</h1>
<p>Мой первый параграф.</p>

<script>
document.write(5 + 6);
</script>

</body>
</html>

Использование window.alert()

Вы можете использовать окно предупреждения для отображения данных:

<!DOCTYPE html>
<html>
<body>

<h1>Моя первая веб страница</h1>
<p>Мой первый параграф</p>

<script>
window.alert(5 + 6);
</script>

</body>
</html>

Вы можете пропустить window ключевое слово.

<!DOCTYPE html>
<html>
<body>

<h1>Моя первая веб страница</h1>
<p>Мой первый параграф</p>

<script>
alert(5 + 6);
</script>

</body>
</html>

Использование console.log()

В целях отладки вы можете вызвать console.log() метод в браузере для отображения данных.

<!DOCTYPE html>
<html>
<body>

<script>
console.log(5 + 6);
</script>

</body>
</html>

]

JavaScript не имеет объекта печати или методов печати.
Вы не можете получить доступ к устройствам вывода из JavaScript.
Единственное исключение - вы можете вызвать window.print() метод в браузере, чтобы распечатать содержимое текущего окна.

<!DOCTYPE html>
<html>
<body>

<button onclick="window.print()">Распечатайте эту страницу</button>

</body>
</html>
