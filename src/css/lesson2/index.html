CSS Границы

CSS свойства border позволяют задать стиль, ширину и цвет границ элемента.

  • dotted - Определяет границу штрихом
  • dashed - Определяет границу пунктиром
  • solid - Определяет границу сплошной
  • double - Определяет границу двойной
  • groove - Определяет 3D границу желобом. Эффект зависит от значение border-color
  • ridge - Определяет 3D границу коньком. Эффект зависит от значение border-color
  • inset - Определяет 3D границу вставкой. Эффект зависит от значение border-color
  • outset - Определяет 3D границу начальной. Эффект зависит от значение border-color
  • none - Определяет без границ
  • hidden - Определяет скрытые границы
p.dotted {border-style: dotted;}
p.dashed {border-style: dashed;}
p.solid {border-style: solid;}
p.double {border-style: double;}
p.groove {border-style: groove;}
p.ridge {border-style: ridge;}
p.inset {border-style: inset;}
p.outset {border-style: outset;}
p.none {border-style: none;}
p.hidden {border-style: hidden;}
p.mix {border-style: dotted dashed solid double;}

CSS Ширина границ

Свойство border-width задает ширину всех четырех границ.

Ширина может быть установлена размером в ( px, pt, cm, em, т.п.) или с помощью одного из трех предопределенных значений:thin (тонкий), medium (средний), или thick (толстый).

p.one {
  border-style: solid;
  border-width: 5px;
}

p.two {
  border-style: solid;
  border-width: medium;
}

p.three {
  border-style: dotted;
  border-width: 2px;
}

p.four {
  border-style: dotted;
  border-width: thick;
}

Определенная ширина границы

Свойство border-width может иметь от одного до четырех значений (для верхней границы, правой границы, нижней границы и левой границы):

p.one {
  border-style: solid;
  border-width: 5px 20px; /* 5px сверху и снизу, 20px по бокам */
}

p.two {
  border-style: solid;
  border-width: 20px 5px; /* 20px сверху и снизу, 5px по бокам */
}

p.three {
  border-style: solid;
  border-width: 25px 10px 4px 35px; /* 25px сверху, 10px справа, 4px снизу и 35px слева */
}

CSS Цвет границ

Свойство border-color используется для установки цвета четырех границ.

  • Название - укажите название цвета, например "red"
  • HEX - укажите значение HEX, например "#ff0000"
  • RGB - укажите значение RGB, например "rgb(255,0,0)"
  • HSL - укажите значение RGB, например "hsl(0, 100%, 50%)"
  • Прозрачный

p.one {
  border-style: solid;
  border-color: red;
}

p.two {
  border-style: solid;
  border-color: green;
}

p.three {
  border-style: dotted;
  border-color: blue;
}

Определенный цвет границ

Свойство border-color может иметь от одного до четырех значений (для верхней границы, правой границы, нижней границы и левой границы):

p.one {
  border-style: solid;
  border-color: red green blue yellow; /* красный сверху, зеленый справа, синий снизу и желтый слева */
}

Цвет HEX

Цвет границы также можно задать с помощью шестнадцатеричного значения (HEX):

p.one {
  border-style: solid;
  border-color: #ff0000; /* красный */
}

Цвет RGB

Или с помощью значений RGB:

p.one {
  border-style: solid;
  border-color: rgb(255, 0, 0); /* красный */
}

Цвет HSL

p.one {
  border-style: solid;
  border-color: hsl(0, 100%, 50%); /* красный */
}

CSS Border - отдельные стороны

Из примеров на предыдущих страницах учебника вы узнали, что для каждой стороны можно задать разные границы.
В CSS также есть свойства для указания каждой из границ отдельно (верхняя, правая, нижняя и левая):

p {
  border-top-style: dotted;
  border-right-style: solid;
  border-bottom-style: dotted;
  border-left-style: solid;
}

}

Поля - сокращенное свойство

Чтобы сократить код, можно указать все свойства полей в одном свойстве.

p {
   margin: 100px 150px 100px 80px;
}

  • margin: 25px 50px 75px 100px;
    • поле сверху 25px
    • поле справа 50px
    • поле снизу 75px
    • поле слева 100px

p {
  margin: 25px 50px 75px;
}

  • margin: 25px 50px 75px;
    • поле сверху 25px
    • поле справа и слева 50px
    • поле снизу 75px

p {
  margin: 25px 50px;
}

  • margin: 25px 50px;
    • поле сверху и снизу 25px
    • поле справа и слева 50px

p {
  margin: 25px;
}

  • margin: 25px;
    • поля со все четырех сторон 25px


Значение авто

Вы можете установить для свойства margin значение auto элемент будет находится в центре внутри контейнера по горизонтали.

Элемент примет положение указанной ширины, а остальное поле будет разделена поровну, между левым и правым полями:

div {
   width: 300px;
   margin: auto;
   border: 1px solid red;
}

Значение наследовать

Пример ниже позволяет полю слева, быть унаследованным от родительского элемента:

div.container {
   border: 1px solid red;
   margin-left: 100px;
}

p.one {
   margin-left: inherit;
}

Свернуть поля

Верхние и нижние поля элементов иногда сворачиваются в одно поле, равное самому большому из двух полей.
Это не происходит на левом и правом полях! Только верхнее и нижнее поля!
Рассмотрим следующий пример:

h1 {
  margin: 0 0 50px 0;
}

h2 {
  margin: 20px 0 0 0;
}

В приведенном выше примере, элемент <h1> имеет нижнюю границу 50px и элемент <h2> имеет верхнее поля значение 20px.

Здравый смысл, казалось бы, предполагает, что вертикальная граница между <h1> и <h2> будет в общей сложности 70px (50px + 20px). Но из-за краха маржи фактическая маржа в конечном итоге составляет 50px.

Отступ и ширина

CSS свойство width определяет ширину области содержимого элемента. Область содержимого, это часть находится внутри элементов padding, border и margin.

div {
