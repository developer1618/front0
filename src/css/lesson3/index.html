CSS Высота и ширина

CSS Свойства height и width используются для установки высоты и ширины элемента.

CSS Свойство max-width используется для установки максимальной ширины элемента.


CSS Примеры высоты и ширины
div {
  height: 200px;
  width: 50%;
  background-color: powderblue;
}

Этот элемент имеет высоту 200 пикселей и ширину 50%

CSS Бокс модель


Все HTML элементы могут рассматриваться как блоки. Термин в CSS "бокс модель" используется, когда речь идет о дизайне и верстке.
Блочная модель CSS - по сути коробка, которая обертывает каждый элемент HTML. Она состоит из: полей, границ, отступов и контента. На рисунке ниже показана модель коробки:

  • Контент - Содержимое коробки, где появляются текст и изображения
  • Отступы - Очищает область вокруг содержимого. Отступы прозрачные
  • Границы - Граница, которая идет вокруг отступа и содержания
  • Поля - Очищает область за пределами коробки. Поля прозрачные

div {
   width: 300px;
   border: 25px solid green;
   padding: 25px;
   margin: 25px;
}

Outline - это линия, проведенная за пределами границы элемента.

CSS Свойство outline

Свойство outline - это линия, которая рисуется вокруг элементов, ЗА ПРЕДЕЛАМИ границ, чтобы элемент "выделялся".


  • outline-style
  • outline-color
  • outline-width
  • outline-offset
  • outline

CSS Ширина контура

Свойство outline-width задает ширину контура и может иметь одно из следующих значений:

  • Тонкий (обычно 1px)
  • Средний (обычно 3px)
  • Толстый (обычно 5px)
  • Конкретный размер в ( px, pt, cm, em, и т.д.)

p.ex1 {
  border: 1px solid black;
  outline-style: solid;
  outline-color: red;
  outline-width: thin;
}

p.ex2 {
  border: 1px solid black;
  outline-style: solid;
  outline-color: red;
  outline-width: medium;
}

p.ex3 {
  border: 1px solid black;
  outline-style: solid;
  outline-color: red;
  outline-width: thick;
}

p.ex4 {
  border: 1px solid black;
  outline-style: solid;
  outline-color: red;
  outline-width: 4px;
}

CSS outline-color

Свойство outline-color используется для установки цвета контура.

p.ex1 {
  border: 2px solid black;
  outline-style: solid;
  outline-color: red;
}

p.ex2 {
  border: 2px solid black;
  outline-style: dotted;
  outline-color: blue;
}

p.ex3 {
  border: 2px solid black;
  outline-style: outset;
  outline-color: grey;
}

HEX Значение

Цвет контура также можно задать с помощью шестнадцатеричного значения (HEX):

p.ex1 {
  outline-style: solid;
  outline-color: #ff0000; /* красный */
}

RGB Значение

p.ex1 {
  outline-style: solid;
  outline-color: rgb(255, 0, 0); /* красный */
}

HSL Значение

p.ex1 {
  outline-style: solid;
  outline-color: hsl(0, 100%, 50%); /* красный */
}


CSS Контур стенография

Свойство outline - это сокращенное свойство для настройки следующих отдельных свойств контура:

p.ex1 {outline: dashed;}
p.ex2 {outline: dotted red;}
p.ex3 {outline: 5px solid yellow;}
p.ex4 {outline: thick ridge pink;}

CSS Смещение контура

Свойство outline-offset добавляет пробел между контуром и краем/границей элемента. Пространство между элементом и его контуром является прозрачным.

p {
  margin: 30px;
  border: 1px solid black;
  outline: 1px solid red;
  outline-offset: 15px;
}

Этот параграф имеет контур размером 15 пикселей за пределами границы.

p {
  margin: 30px;
  background: yellow;
  border: 1px solid black;
  outline: 1px solid red;
  outline-offset: 15px;
}

Этот параграф имеет контур размером 15 пикселей за пределами границы.
