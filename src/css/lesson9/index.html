CSS Комбинаторы

Комбинатор - это то, что объясняет взаимосвязь между селекторами.

  • селектор потомок пространство
  • селектор ребенок >
  • селектор смежный брат +
  • селектор общий брат ~

Селектор потомок

Селектор потомок сопоставляется всем элементам, которые являются потомками указанного элемента.

div p {
   background-color: yellow;
}

Селектор ребенок

Селектор ребенок выбирает все элементы, которые являются непосредственными детьми указанного элемента.

div > p {
   background-color: yellow;
}

Селектор соседний брат

Селектор соседний брат выбирает все элементы, которые являются смежными элементами указанного элемента.
Дочерние элементы должны иметь тот же родительский элемент, "соседний" значит "сразу же после".

div + p {
   background-color: yellow;
}

Селектор общий брат

Селектор общий брат выбирает все элементы, которые являются элементами указанного элемента.

div ~ p {
   background-color: yellow;
}

CSS Псевдо-классы

Псевдо-класс используется для определения особого состояния элемента.

  • Стиля элемента при наведении курсора мыши
  • Стиля посещенных и непосещенных ссылок
  • Стиля элемента, для получения фокуса
/* непросмотренная ссылка */
a:link {
   color: #FF0000;
}

/* просмотренная ссылка */
a:visited {
   color: #00FF00;
}

/* наведение мыши на ссылку */
a:hover {
   color: #FF00FF;
}

/* активная ссылка */
a:active {
   color: #0000FF;
}

CSS Псевдо-элементы

Псевдо-элемент используется для оформления указанной части элемента.

  • Стиля первой буквы или линии элемента
  • Вставки содержимого до или после элемента
p::first-line {
   color: #ff0000;
   font-variant: small-caps;
}

h1::before {
   content: url(smiley.gif);
}

h1::after {
   content: url(smiley.gif);
}

CSS Правило !important

Правило !important в CSS используется для придания большей важности свойству/значению, чем обычно.

#myid {
  background-color: blue;
}

.myclass {
  background-color: gray;
}

p {
  background-color: red !important;
}

CSS Математические функции

Math функции CSS позволяют использовать математические выражения в качестве значений свойств. На данной странице вы узнаете о функциях calc(), max() и min().

Функция calc() выполняет вычисление, которое будет использоваться в качестве значения свойства.

#div1 {
  position: absolute;
  left: 50px;
  width: calc(100% - 100px);
  border: 1px solid black;
  background-color: yellow;
  padding: 5px;
}

Функция max() использует наибольшее значение из списка значений, разделенных запятыми, в качестве значения свойства.

#div1 {
  background-color: yellow;
  height: 100px;
  width: max(50%, 300px);
}

Функция min() использует наименьшее значение из списка значений, разделенных запятыми, в качестве значения свойства.

#div1 {
  background-color: yellow;
  height: 100px;
  width: min(50%, 300px);
}
