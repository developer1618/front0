CSS3 Изображение граница

CSS свойству border-image, можно задать изображение, которое будет использоваться в качестве границы вокруг элемента.

#borderimg {
    border: 10px solid transparent;
    padding: 15px;
    border-image: url(border.png) 30 round;
}

Поддержка браузера для 2D трансформации

  • translate()
  • rotate()
  • scaleX()
  • scaleY()
  • scale()
  • skewX()
  • skewY()
  • skew()
  • matrix()
div {
  transform: translate(50px, 100px);
}

Метод scaleX()

Метод scaleX() увеличивает или уменьшает ширину элемента.

div {
  transform: scaleX(2);
}

Метод scaleY()

Метод scaleY() увеличивает или уменьшает высоту элемента.

div {
  transform: scaleY(3);
}

Метод skewX()

Метод skewX() наклоняет элемент вдоль оси X на заданный угол.

div {
  transform: skewX(20deg);
}

Метод skewY()

Метод skewY() наклоняет элемент вдоль оси Y на заданный угол.

div {
  transform: skewY(20deg);
}

Метод skew()

Метод skew() наклоняет элемент вдоль осей X и Y на заданные углы.

div {
  transform: skew(20deg, 10deg);
}

Метод matrix()

Метод matrix() объединяет все 2D методы преобразования в один.

div {
   transform: matrix(1, -0.3, 0, 1, 0, 0);
}

Метод rotateY()

Метод rotateZ()

CSS3 Маскировка

С помощью CSS маскировки вы создаете слой-маску для размещения поверх элемента, чтобы частично или полностью скрыть часть элемента.

.mask1 {
  -webkit-mask-image: url(sw3logo.png);
  mask-image: url(sw3logo.png);
  -webkit-mask-repeat: no-repeat;
  mask-repeat: no-repeat;
}

CSS3 Пагинация

Если у вас есть веб сайт с большим количеством страниц, вы можете добавить своего рода нумерация страниц на каждой странице:

.pagination {
    display: inline-block;
}

.pagination a {
    color: black;
    float: left;
    padding: 8px 16px;
    text-decoration: none;
}
